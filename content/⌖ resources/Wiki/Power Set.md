## Power Set

Power set of set $C$, $S(C)$ is **the** set of all subsets of $C$, including **the** empty set $\varnothing$ and the set $C$ itself.

[[Cardinality]] of a power set $S(C)$ is the square of the [[Cardinality]] of the set itself.

$$|S(C)| = |C|^2$$
