## Metadata
- Association: [[Set Theory]], [[Power Set]]
- Domain: [[Domain Root - Mathematics]]

Denoted by an absolute-value symbol, cardinality of a set is the number of unique elements contained within it. For example:

$A = {1, 2, 3}; | A | = 3$
$B = {1, 2}; | B | = 2$
$C = {1}; | C | = 1$
